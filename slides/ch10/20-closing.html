Run the three-question read on a brand you actually buy.

01 What is the buyer receiving? Name the value against the next-best option, and the evidence you have about what they would pay.
02 What comparison will they use? Find the reference set the buyer judges the price against before the number even registers.
03 What rule explains the number? State why this buyer pays this amount now, then test whether it survives a screenshot going viral.
04 Hand it to an assistant. Ask which approach fits and a band, then read where the cheap tier stops and the deep tier names the levers.
