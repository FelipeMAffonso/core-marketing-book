Hold the five parts and the rest of the book has its floor.

The file Read in chunks, where a chunk past the window never arrives. The warning has to be you.
The wire The trip to a metered server and back. Store the key under a name, never in a file you share.
The model A well-read autocomplete that searches the web yet can still be confidently wrong.
The agent That model in a loop with a few tools and a stop button, working a goal you state once.
The stack The four-rung climb, from a chat box to a Monday agent that runs while you sleep.
