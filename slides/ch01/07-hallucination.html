The model reaches for the most natural-sounding next word, not the most truthful one.

Most of the time the two are the same word, which is exactly why the model feels trustworthy. When they part ways it states something false in the same calm voice it uses for the truth.

Letting it search the web cuts this down without ending it, so the work rule holds: every number, name, and date gets a real source clicked first. The cheapest check is the link itself.
