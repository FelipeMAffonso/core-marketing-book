Canvas / LMS Common Cartridge

This is a Common Cartridge (.imscc) export of the course test bank, packaged as IMS QTI so it imports cleanly into Canvas, Blackboard, Moodle, and other LMS platforms that accept Common Cartridge. Importing it drops the question bank into your course shell as ready-to-use quiz items.

Download

Download principles-of-marketing-testbank.imscc · 479 KB

How to import (Canvas)

  1. In your Canvas course, open Settings → Import Course Content.
  2. Choose Common Cartridge 1.x Package as the content type.
  3. Upload the principles-of-marketing-testbank.imscc file and run the import.
  4. The test-bank items appear under Question Banks / your quizzes once the import completes.

The cartridge is generated from the same 27-chapter test bank rendered under Materials → Test & Quiz Bank. This is a draft export; review items in your LMS before publishing to students.
